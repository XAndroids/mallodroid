mallodroid
==========
在Android应用中查找有问题的SSL证书验证。MalloDroid是在androguard (https://code.google.com/p/androguard/) 基础之上构建的，Android应用程序逆向工程框架小工具。因此，MalloDroid需要使用androguard.

===============================

### 使用
例子: ./mallodroid.py -f ExampleApp.apk -x

./mallodroid.py --help

===============================
